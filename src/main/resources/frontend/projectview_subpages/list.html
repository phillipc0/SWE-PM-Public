Auf der Ansicht Liste werden die Aufgaben in Listenform dargestellt. Eine Unteraufgabe steht immer unter der Hauptaufgabe. Sie können nach den Spalten sortieren und nach fertigen oder eigenen Aufgaben filtern. Per Klick auf die Zeile einer Aufgabe werden die Details angezeigt. Jede Aufgabe kann in einen Kalender exportiert oder als ICS-Datei heruntergeladen werden.
