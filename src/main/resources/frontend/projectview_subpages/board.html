Auf der Ansicht Board werden die Aufgaben in der Spalte des jeweiligen Status angezeigt. Sie können entweder alle Aufgaben oder die Ihnen zugewiesenen Aufgaben anzeigen.
Mithilfe des HyperMove™ Systems können Aufgaben auch auf kleinen Bildschirmen einfach per Drag-and-Drop verschoben werden.
Per Klick auf eine Aufgabe werden die Details angezeigt. Durch Hovern über eine Aufgabe werden Abhängigkeiten farblich dargestellt.
Rot = Die aktuelle Aufgabe wird von der roten Aufgabe blockiert
Grün = Die in grün markierten Aufgaben sind Unteraufgaben der aktuellen Aufgabe
Blau = Die aktuelle Aufgabe ist eine Unteraufgabe der blau markierten Aufgabe
